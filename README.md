# TPU High-Performance Computing (HPC)

TPU HPC enables high-performance scientific and technical computing and
multiphysics simulation efficiently and at scale using Google's Tensor
Processing Units (TPUs). The goal is to develop specialized programs that
empower TPU hardware to run complex scientific simulations efficiently.

## Installation

To run TPU HPC benchmarks, you will need access to
[TPUs on Google Cloud](https://cloud.google.com/tpu/docs/tpus). For small
simulations, the easiest way to access TPUs is to use
[Google Colab](https://colab.research.google.com/) with a TPU runtime.
