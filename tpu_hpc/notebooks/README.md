# TPU HPC Notebooks

This folder contains interactive Colab notebooks and presentation materials
for the [TPU High-Performance Computing (HPC)](../README.md) project. These
resources are intended to help external partners and collaborators get started
with TPU-based scientific computing.

## Getting Started

All notebooks in this folder are designed to run on
[Google Colab](https://colab.research.google.com/) with a TPU runtime.

To open a notebook:

1. Go to [colab.research.google.com](https://colab.research.google.com/).
2. Select **File → Open notebook → GitHub**.
3. Enter the repository URL:
   `https://github.com/google-research/tpu-hpc`
4. Navigate to the desired notebook under `notebooks/`.
